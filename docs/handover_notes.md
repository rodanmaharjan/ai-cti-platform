# Handover Notes

## Project Title

AI-Assisted IOC Analysis Tool

---

# Project Overview

This project is a simple cybersecurity dashboard developed using Python and Streamlit. The tool analyses Indicators of Compromise (IOCs) such as IP addresses, domains, and file hashes.

The system classifies IOC types, assigns risk levels, maps techniques to the MITRE ATT&CK framework, and generates investigation recommendations.

---

# Main Features

- IOC type detection
- Multi-IOC analysis
- MITRE ATT&CK mapping
- Risk classification
- Recommendation generation
- CSV report export
- Streamlit dashboard interface
