# AI-Assisted IOC Analysis Tool

## Overview
This project is a simple AI-assisted cybersecurity dashboard developed using Python and Streamlit. The tool analyses Indicators of Compromise (IOCs) such as IP addresses, domains, and file hashes.

## Features
- IOC type detection
- MITRE ATT&CK mapping
- Risk level classification
- CSV report export
- Multi-IOC analysis

## Technologies Used
- Python
- Streamlit
- Pandas
- GitHub
