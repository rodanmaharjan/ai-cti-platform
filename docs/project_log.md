# Project Log

## Project Overview

This project focused on developing a simple AI-assisted IOC (Indicator of Compromise) analysis dashboard for cybersecurity analysis and reporting. The system was developed incrementally using Python, Streamlit, and GitHub.

# Key Development Milestones

## Milestone 1 — Initial Prototype

The first working prototype included:
- basic Streamlit dashboard,
- IOC input field,
- and simple output display.

This stage confirmed that the application environment was working correctly.

---

## Milestone 2 — IOC Analysis Features

The second stage focused on implementing cybersecurity functionality, including:
- IOC type detection,
- MITRE ATT&CK mapping,
- risk classification,
- and recommendation generation.

This stage transformed the project from a basic dashboard into a simple cybersecurity analysis tool.

---

## Milestone 3 — Reporting and UI Improvements

The third stage focused on improving usability and presentation quality. CSV export functionality was added to allow report generation, and the Streamlit interface was improved to make results easier to read and understand.

---

## Milestone 4 — Documentation and Repository Management

The final stage focused on project organisation and documentation. GitHub was used for version control, and supporting documentation was created to explain the development workflow, methodology, and AI-assisted processes used during implementation.

---

# Reflection

The project demonstrated how AI-assisted workflows can support rapid cybersecurity tool development. The use of Generative AI tools improved development speed, debugging, and code understanding throughout the project lifecycle.

The incremental prototyping approach allowed the project to remain functional throughout development while gradually introducing additional cybersecurity analysis features.
